# OPTIMIZACION_FLUJO_CORREOS
Sistema de automatización basado en n8n e Inteligencia Artificial (Gemini) diseñado bajo el marco de la Cognición Extendida. El asistente actúa como un sistema de memoria externa que monitorea el correo institucional, filtra información irrelevante, extrae fechas límite de entrega y las sincroniza proactivamente en Google Calendar y Telegram.
